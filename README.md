# 2048

Classic 2048 tile game in the browser.

## Play online

**https://potat2201.github.io/2048/**

> GitHub Pages hosts the game at the link above.  
> `https://github.com/potat2201/2048` is the source repository, not the playable site.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Controls

- Arrow keys or WASD to move
- Swipe on touch devices
- **New Game** to restart
